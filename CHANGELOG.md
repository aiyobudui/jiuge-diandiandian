# 更新日志

## v1.2.0 (2026-06-13)

### Bug 修复
- 修复编辑动作对话框点击「目标按键」崩溃（NullReferenceException，`pnlKeyHighlight` 局部变量未赋值类字段）
- 修复编译时 `diag.cs` 临时调试文件导致报错
- 修复 `KeyRecordDialog` 布局问题（窗口过小、控件未居中）
- 修复 ICO 图标在资源管理器中显示模糊（全 PNG 格式兼容性差，改为 BMP 16-128px + PNG 256px 混合格式）
- 修复编译脚本双击闪退（UTF-8 编码在中文 Windows 不兼容，改为 GBK 编码）
- 修复 `dotnet publish` 后生成多余 `Release\net10.0-windows` 文件夹（编译脚本中自动清理）

### 新增
- 配置文件支持**自定义文件名**保存和加载，支持格式验证
- 新增 `编译.bat` 一键编译脚本（自动清理 → 还原 → 发布单文件 exe → 清理多余文件夹）
- 新增 `ConfigService.SaveToFile()` / `ConfigService.LoadFromFile()` 方法

### 优化
- **UI 优化**：DataGridView 所有列内容居中显示
- **UI 优化**：双击动作列表空白区域直接打开「添加动作」对话框
- **UI 优化**：编辑动作对话框中「目标按键」区域独特显示（蓝色粗体标签 + 浅黄色外框 + 红褐色文字）
- 编译输出路径统一到 `bin\九歌键鼠助手.exe`
- .NET 版本升级至 .NET 10

## v1.1.0 (2026-06-12)

### 改名
- 项目显示名由「九歌按键点点点」更名为「九歌键鼠助手」

### Bug 修复
- 修复编辑动作对话框数据丢失问题（改为 Clone 拷贝）
- 修复快捷键 Ctrl+Shift+Z 不生效（`Keys.C` → `Keys.Z`）
- 修复动作执行器线程安全问题（`_isRunning` 加 `volatile`，`Stop()` 加 `Thread.Join`）
- 修复动作执行额外延迟重复累加
- 修复配置保存时 `LoopInterval` 残留导致异常
- 修复托盘退出时 NotifyIcon 已释放导致崩溃
- 修复启动时双重气泡提示冲突

### 优化
- 系统托盘最小化/恢复逻辑完善
- 窗口关闭处理优化（区分系统关机和用户关闭）
- 状态栏访问简化，移除递归搜索
- 删除动作增加确认对话框
- 抓取坐标功能改为快捷键提示标签
- 清理未使用的字段和包引用
- Mutex 加 `Local\` 前缀防跨会话冲突
- 列名调整：前延迟 → 前间隔毫秒，间隔毫秒 → 后间隔毫秒
- 启动时托盘气泡提示"程序已启动"
- 添加 README.md 和 CHANGELOG.md
