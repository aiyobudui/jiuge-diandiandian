# 九歌键鼠助手

> 一款轻量级键盘鼠标宏工具，支持录制、编辑、循环执行键鼠动作序列。

---

## 功能概述

- **键鼠动作录制**：支持键盘按键、鼠标左/中/右键单击
- **动作序列编辑**：可视化列表管理，支持添加、编辑、删除、排序
- **循环执行**：每个动作支持单次 / 循环 / 直到手动停止三种模式
- **全局快捷键**：默认 `F8` 一键启停，可自定义
- **系统托盘**：最小化到托盘后台运行，气泡通知状态变化
- **配置持久化**：保存/导入配置文件（INI 格式），支持自定义文件名，启动自动加载
- **坐标定位**：鼠标点击动作支持精确坐标（Ctrl+Shift+Z 抓取）

---

## 动作类型

| 类型 | 说明 |
|------|------|
| 键盘按键 | 模拟单个按键（支持 Ctrl/Shift/Alt 组合） |
| 鼠标左键 | 在指定坐标单击左键 |
| 鼠标右键 | 在指定坐标单击右键 |
| 鼠标中键 | 在指定坐标单击中键 |

## 动作属性

| 属性 | 说明 |
|------|------|
| 重复次数 | 该动作重复执行的次数 |
| 前间隔毫秒 | 动作执行前的等待时间（毫秒） |
| 后间隔毫秒 | 动作执行后的等待时间（毫秒） |
| 鼠标坐标 | 鼠标点击的目标坐标（仅鼠标动作） |
| 备注 | 自定义备注文字 |

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `F8`（默认） | 启动/停止动作序列 |
| `Ctrl+Shift+Z` | 抓取鼠标当前坐标 |

> 启动/停止快捷键可在主界面点击「自定义修改」更换。

---

## 使用步骤

1. 点击 **添加动作**，选择动作类型并设置参数
2. 按需调整动作顺序（上移/下移）、修改重复次数和延迟
3. 点击 **保存配置** 持久化当前动作列表（支持自定义文件名）
4. 按 `F8` 或点击 **手动启动** 开始执行
5. 再次按 `F8` 或点击 **手动停止** 停止

## 系统要求

- **操作系统**：Windows 10 / 11
- **运行时**：.NET 10 Desktop Runtime
- **权限**：无需管理员权限

## 构建

```bash
# 使用 编译.bat 一键发布（推荐）
编译.bat

# 或手动编译
dotnet publish -c Release -r win-x64 --self-contained true -o bin
```

输出文件：`bin\九歌键鼠助手.exe`

> 编译.bat 会自动清理临时文件、还原依赖、发布单文件 exe，并清理多余的输出文件夹。

## 配置文件

程序使用 INI 格式配置文件，默认读取同目录下的 `config.ini`。

支持**另存为**自定义文件名，也支持**加载**任意 INI 配置文件（格式验证）。

配置文件格式示例：

```ini
[Settings]
StartKey=F8

[Actions]
Count=1

[Action0]
Type=Keyboard
Key=Ctrl+C
RepeatCount=1
PreDelay=0
Delay=50
Comment=复制
```
